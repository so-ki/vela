★ 相对竞品，我们刻意做的差异化（功能 + 技术组合）

  • 项目制：北大法宝强在「查法条」；我们强在「一个投资项目走完全流程」
  • 巴西垂直 + 可引用：每条 AI 结论尽量绑语料片段/URN，低分拦 S3，不假装有依据
  • 可签：Gate A + 逐条复核 + 定稿后法条不 silently 变——给法务部敢用的失败安全
  • 统摄 + 对抗：不只生成正文，还用 Red Team 专门找材料/协查/合同之间的矛盾
