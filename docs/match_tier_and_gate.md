# 匹配度评分与 S1/S2/S3 门控说明

> 面向评委 / 技术审：解释「70 分门控」如何计算、何时硬阻断、与 Gate A 的关系。  
> 实现代码：`backend/app/services/match_tier_service.py`、`legal_rag.py`

## 1. 产品定位

Vela **不是**「自动出具法律意见」的引擎，而是 **协查辅助工具**：

- 为每条核查清单条目检索 Top-3 法条片段（精选语料 + 可选 Chroma）
- 给出 **可解释的匹配度分数**（关键词 + 维度 + 清单编号加权）
- 按 **S1 / S2 / S3** 分级，低于有效阈值的条目 **不得自动定稿**

## 2. 匹配度如何计算（MVP）

| 因素 | 权重说明 |
|------|----------|
| 清单条目编号命中语料 `checklist_codes` | +45 分 |
| 审查维度与语料 `dimension` 一致 | +20 分 |
| 查询词与 tags / 标题 / 正文（清洗后）重叠 | 每个词 +8 分 |
| Chroma 向量检索（若已建索引） | 距离转分 + 清单编号 boost，与关键词结果合并取 Top |

**说明：**

- 分数上限 100，**不是**可审计的语义相似度模型；演示环境以 **关键词 + 规则加权** 为主。
- 语料 `text_pt` 已清洗 HTML/JS 噪声；展示优先 **中文摘要 `text_zh` + 短葡语摘录**，外链 LexML/Planalto 供人工核对全文。
- 可选 Chroma 为辅助召回，不改变 S1/S2/S3 门控逻辑。

## 3. 有效阈值（Effective Threshold）

法务在确认协查范围时可设置 **基础阈值**（50–95，**默认 70**）。  
系统还会按条目 **优先级** 抬高下限：

| 优先级 | 有效阈值下限 |
|--------|--------------|
| high | max(基础阈值, 75) |
| medium | max(基础阈值, 70) |
| low | max(基础阈值, 65) |

支持按条目 **± 调整**（用户偏好），仍限制在 50–95 之间。

## 4. S1 / S2 / S3 分级规则

| 等级 | 标签 | 条件（摘要） | 对导出/定稿 |
|------|------|--------------|-------------|
| **S1** | 自动通过 | 有命中；匹配度 ≥ 有效阈值；grounding 通过 | 可纳入自动简报，仍建议法务抽检 |
| **S2** | 人工复核 | 有命中但分数接近或略低于阈值；或非 high 优先级略低 | 必须标注「需法务复核」 |
| **S3** | 硬阻断 | 无命中；或 high 优先级未 grounding；或 high 且分数 < 阈值−15 | **gate_status=blocked**，不可当作已核实 |

典型 S3 原因：

- 「未检索到法条片段」
- 「高优先级条目引证未通过 grounding 校验」
- 「高优先级匹配度低于硬阻断线（阈值 − 15）」

## 5. 与 Gate A、法务复核的关系

```
场景提交 → 清单生成 → RAG 检索 + 匹配度
                              ↓
                    每条 S1/S2/S3 分级
                              ↓
              Gate A（材料要件 + 聚合门控）→ 锁定/放行清单复核
                              ↓
              法务逐条确认 / 驳回 → 全部处理 → 定稿 v1.x
                              ↓
                    Word/PDF 导出（含免责声明）
```

- **Gate A**：材料构成 + 匹配度聚合；未通过则清单复核锁定（见 `compliance_and_security.md`）。
- **法务复核**：S1 条目也可被驳回；S2/S3 必须人工处理后才能定稿。
- **LLM 润色**：仅改措辞，**不新增**法条、结论或溯源链接。

## 6. MVP 边界（诚实声明）

| 已做 | 未做 / 演示级 |
|------|----------------|
| ~70 条精选语料 + URN 外链 | LexML **全库**实时 API |
| 关键词 + 可选 Chroma | 大规模标注集、命中率 ≥85% 证明 |
| S1/S2/S3 可解释门控 | 外聘律师抽检报告 |
| 本地语料指纹监测 + 手动扫描 | 7×24 自动爬虫订阅 |
